## 搬瓦工 VPS 是什么，为什么价格差距这么大

搬瓦工（BandwagonHost）是加拿大 IT7 Networks 旗下做了二十年的 VPS 服务商，全称 BandwagonHost，因为发音跟"搬瓦工"相近，国内就这么叫开了。VPS 本质上就是一台在海外数据中心运行的云服务器，你可以远程登录、部署服务、建网站，或者做一些需要境外网络访问的事情。

搬瓦工套餐贵，贵在哪？不同线路。

普通国际线路的套餐，年付五六十美元；走电信 CN2 GIA 高端线路的，年付要一百七八美元；到了香港 CN2 GIA，月付就接近九十美元。线路不同，带来的延迟、稳定性、高峰期速度差距非常大——这不是玄学，而是物理距离和网络资源投入的现实。

所以"搬瓦工好贵"这个感受，通常来自两种情况：一是拿高端线路的价格跟别家普通线路比，二是自己需求没那么高但看到了贵的套餐。

## 觉得贵？四个方向可以省

### 一、先搞清楚自己需要哪条线路

这是省钱效率最高的一步。不少人说搬瓦工贵，是因为默认看的是 CN2 GIA-E 套餐。但如果你的用途只是学习 Linux、跑个简单的程序、偶尔访问一下，KVM 套餐 $49.99/年完全够用，算下来每天不到一块四毛钱。

这两者的速度差距，晚高峰会比较明显——CN2 GIA-E 走的是高端优化线路，高峰期稳定性好很多。但如果你用 VPS 的场景主要在白天、流量不大，KVM 套餐能省下一百二十美元/年，性价比完全没问题。

选错线路才是最贵的事。
